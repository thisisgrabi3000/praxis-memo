Praxis Memo · Update-Ablauf

Wie Updates nach der Installation laufen — für Miriam und für CMG.

Für Miriam — So macht sie ein Update

1
Doppelklick auf „Praxis Memo Update" am Desktop Schwarzes Fenster geht auf — alles läuft automatisch.
2
Warten (2–10 Minuten) Sie kann nebenbei was anderes machen. Das Skript redet ständig was es gerade tut.
3
„Update fertig" → Taste drücken zum Schließen Beim nächsten Start von „Praxis Memo" ist die neue Version aktiv.
Wie oft? Einmal im Monat reicht. Oder wenn CMG anruft: „Klick mal Update."

Was die Update-BAT im Hintergrund macht

SchrittWas passiert
1Backup: Patientendaten + alte App-Dateien werden in backups\vor-update\ mit Zeitstempel gesichert. Vorher passiert nichts Riskantes.
2App-Code-Update: curl lädt das neueste ZIP von GitHub Releases. Wird entpackt und überschreibt app.js, index.html, styles.css, praxis_memo_server.py. Bei Fehler: Rollback aus Backup, BAT läuft weiter.
3Python check. Wenn fehlt → Abbruch mit Hinweis auf „KI einrichten.bat".
4Diktat-KI update: pip install --upgrade faster-whisper. Bei Fehler: alte Version läuft weiter.
5Ollama check + Modell-Pull: ollama pull qwen2.5:3b. Lädt nur Diff, geht meist in Sekunden.
6Desktop-Verknüpfung prüfen — neu anlegen falls Miriam sie aus Versehen gelöscht hat.

Patientendaten in data\ und Backups in backups\ werden nie vom Update angefasst.
